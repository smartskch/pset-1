Problem Set - 1

Problem 1.1. Liar! Liar!

  • Read the below statements and answer.

  1. The number of false statements here is one.
  2. The number of true statements here is two.
  3. The number of false statements here is three.

  4. The number of true statements here is four.

  • Which of the above statements is true?

Problem 1.2. Counterfeit!

  • There is a heap of 8 identical Rs 1 coins. But one of them is fake and varies only in weight-it’s either lighter or heavier than usual. Given an empty beam balance (No standard weights, sorry kids) how will you find the fake coin? Use the least no. of weighing attempts. (Hint: Think of the worst case possible)

Problem 1.3. Magic Shoes!

  • Satish is ‘x’ metres away from a wall. Every step he takes he covers half the remaining distance between him and the wall. In how many steps does he reach the wall?
